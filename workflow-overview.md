# Workflow Overview

CloudMind Writer Flow is based on a two-agent workflow for generating and publishing long-form technical articles on Hashnode.

## Workflow

```text
User idea or topic
        ↓
Agent 1: Publishing Article
        ↓
Complete Hashnode article package
        ↓
Manual review
        ↓
Agent 2: Cover Image Generator
        ↓
16:9 technical cover image
        ↓
Hashnode publishing
```

## Platforms Used

- OpenAI Platform for agent creation
- Web search for article research and validation
- Image generation for Hashnode cover images
- Hashnode for article publishing
- GitHub for project documentation and portfolio presentation

## Main Goal

The goal is to create a repeatable editorial workflow that turns raw ideas into professional technical articles with SEO metadata and cover images.
